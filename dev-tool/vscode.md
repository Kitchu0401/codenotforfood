# Visual Studio Code

## 터미널에서 vscode로 파일 바로 열기

`Ctrl + Shift + P` 로 다음의 쉘-커맨드를 호출하여 `code` 환경변수를 추가한다.
```
> Shell Command: Install 'code' command to in PATH
```

이후 간단하게 다음의 명령어로 내부 터미널에서 파일을 새롭게 생성하거나 편집 할 수 있다.
``` sh
$ code FILE_NAME
```
